Toolbar Features:

Check Mail:
Retrieves new mail sent to your mailbox, and displays those messages within the "In" folder.

New Message:
Opens a "New Message" window where you can compose and send an email message.

Preferences:
Opens a "Preferences" window where you can view/edit your general mailbox settings. You may also add outside email accounts within this area if you'd like to protect multiple mailboxes with your SpamHippo anti-spam/anti-virus filters.

SpamHippo:
SpamHippo protects your mailbox against unwanted spam & viruses. Within this area you can enable and modify a variety of SpamHippo filters such as sender verification, approved senders list, blacklists, and custom filter rules to block spam & viruses. You may also apply your SpamHippo protection to outside mailboxes!

Address Book:
Opens an "Address Book" window where you can create and edit contact lists. This feature is commonly used to link a nickname to a large list of email addresses that you frequently send email to.

Help:
This link offers helpful information about the Direct Read Mail area that you're currently viewing.

Logout:
This link allows you to logout from your mailbox.
 
 Folders Area:

Add New Folder ( ):
To create a new folder, click on this icon within the folders area. Please note that folder names must use alphanumeric characters, and can be up to 10 characters in length.

Empty Trash ( ):
To permanently delete any messages that appear within your "Trash" folder, click this icon within the folders area.

Trash In Folder ( ):
To permanently delete all of the messages that appear within your "In" folder, click this icon within the folders area.

Folder States
When the contents of a folder are being viewed an open folder icon ( ) will appear, and a closed folder icon ( ) will appear next to the rest of the folders within list of folders. When messages are being moved from one folder to another, a folder transfer icon ( ) will appear next to the folder that will be receiving the message transfer. Additionally, if your mouse cursor is placed over a file folder's name, the number of read/unread messages ( ) will be displayed for that particular folder.

In:
New mail messages are received in this folder. When you click the folder name, the folder icon will change to an "open" state, and a list of your received messages will be displayed in the "Folder View" area, which appears to the right of the "Folders" area.

Out:
Outbound mail messages will appear in this folder. When you click the folder name, the folder icon will change to an "open" state, and any messages you have sent will be displayed in the "Folder View" area, which appears to the right of the "Folders" area.

Trash:
Trashed mail messages will appear in this folder. When you click the folder name, the folder icon will change to an "open" state, and any trashed messages will be displayed in the "Folder View" area, which appears to the right of the "Folders" area. To permanently remove messages from your mailbox, you must transfer them to your "Trash" folder and click the "Empty Trash" icon that appears at the top of the folder column. You may also delete the entire contents of your "In" folder by clicking the "Trash In Folder" icon.

Folders Floating Menu:
When you place your cursor over any of the folder icons, a blue floating menu will appear with the following options:

  • Open folder - This will display the contents of a specific folder in the "Folder View" area. Alternately, you can click a folder's title to display its contents.
     

  • Transfer selected - One or more messages can be selected in the "Folder View" area by using your mouse and the Ctrl key on your keyboard. Once the messages have been selected, you can use the "Transfer selected" option to move the selected messages from one folder to another. Alternately, you may transfer selected messages by simply dragging your mouse cursor from the "Folder View" area to the new folder that will receive the transferred messages. As an example you can select unwanted messages within your "In" folder, and then drag/transfer them to your "Trash" folder.
     

  • Rename folder - If you've created a folder within your mailbox, you can use this option to change the name of that folder.
     

  • Delete folder - Allows you to delete a mail folder that you've created. Please note that this option deletes any content that appears in the folder you've chosen to remove.
 
 Folder View Area:

Status:
This column provides a status for each message within your list. If your cursor is placed over a specific menu icon, a status bar will appear and display the size of that email message.

  • Attachment ( ) - A file has been attached to an email message (photo, video, etc.)
     

  • Unread Message ( ) -This indicates that a message has not been read
     

  • Read Message ( ) - This indicates that a message has been read
     

  • Forward Message ( ) - This indicates that you've forwarded a message
     

  • Reply Message ( ) - This indicates that you've replied to a message
     

  • New Message ( ) - This indicates that you've sent a new message to someone

Date:
This column references the date and time that an email message was sent. By clicking on the "Date" heading, you can sort your message list in ascending or descending date order.

From:
This column references the name of the person that sent you an email message. By clicking on the "From" heading, you can sort your message list by alphabetical name order.

Subject:
This column references the subject or title of each message in your list. If a message's subject line is clicked one time, the contents of the message will be displayed in the "Message Preview" area that appears below the "Folder View" area. If you double-click on a message's subject line, the content will be displayed in its own window, from which you can send a reply.

You can also conveniently transfer one or more messages from one folder to another, using Direct Read Mail's drag & drop feature. Using a single mouse click and the Ctrl key on your keyboard, you can select multiple messages from your message list. Once selected, you can drag your mouse cursor over the new folder location where you'd like the messages to be transferred. A folder transfer icon ( ) will appear, and when you release the mouse button the selected messages will be moved to that new folder location.
 
 Message Preview Area:

The "Message Preview" area appears below the "Folder View" area. When you single mouse click on a message within the "Folder View" area, it will display in the "Message Preview" area.

Subject:
This section will display the subject of the email message you are previewing. If you click on the subject the message will be displayed in its own window, from which you can send a reply. You can also open a message in its own window by double mouse clicking on a message's subject within the "Folder View" area.

From:
This section will display the name and address of the person who sent you the email message.

Printable Version ( ):
This button will open the message in a printable format.
 
 Open Message Window:

When a message is opened within a separate window, this screen will appear. A message can be opened by double mouse clicking on its subject line within the "Folder View" area, or clicking on its subject line within the "Message Preview" area.

New Message:
This button opens a separate window that allows you to compose/send a new message.

Reply:
This button opens a separate window that allows you to compose/send a reply message.

Reply All:
This button opens a separate window that allows you to compose/send a reply message to each email address referenced in the From, Cc & Bcc fields within the original email message

Forward:
This button opens a separate window that allows you to forward a message.

View Message / Header / Source:
These buttons allow you to toggle between a message, header and source view.
 
 Composing a Message:

The "New Message", "Reply", "Reply All" and "Forward" buttons will open a separate window that allows you to compose and send a message to someone. The top portion of this window includes the header details for the message, and bottom portion of the window offers an area where you can compose your message.

Send Message:
This button takes your message and sends it to the address referenced in your "To:", "Cc:" and "Bcc:" lines.

Spell Check:
This button will refresh the composition window and display your message in a "Spell Check" mode. If a word has a questionable spelling, it will be displayed within a text box and a question mark icon will appear next to that box. When the question mark icon is selected, a window will display alternate spellings for that word. If you've selected an alternate spelling, it will auto-update within the "Spell Check" window, and you can click the "Save" button to return to the composition window. The "Cancel" button allows you to cancel the "Spell Check" process.

To:
This field indicates the email address your message will be sent to.

From:
This field indicates your email address, which can be edited in the "Preferences" area.

Subject:
This field indicates the subject or title of the message you're sending.

Cc:
This field allows you to send your message to an additional email address.

Bcc:
This field allows you to send your message to a hidden/blind email address, which will not be seen by anyone that receives a copy of your message.

Attachment:
This feature allows you to attach a file (document, photo, video, etc.) to an email message you are sending to someone. Simply click the "Browse" button to locate and attach a file that resides on disk or your local computer.
