Account Details Area:
This area allows you to define and edit the global settings for your email account. The following descriptions further detail each of those settings.

Name:
This is the name that will appear in the "From" line of any messages that you send.

Email Address:
This is the email address that will appear in the "From" line of any messages that you send.

Auto-Check Mail:
This setting determines how frequently your mailbox will automatically check for new email messages. The frequency is defined in terms of minutes, and you can disable the auto-check feature by setting it to "0 minutes".

Image Attachments:
If you've received an email with a graphic or photo attachment (gif, jpeg, etc.), this setting gives you the option of displaying the image within the message area, or displaying the image attachment as hypertext link within the message area.

Message Preview:
This setting gives you the option of showing or hiding the Message Preview window that appears within your main mailbox screen. If you choose to show this area, you can click individual messages within the Folder View area, and have them displayed within a Message Preview window. If you hide this area, the Folder View area will increase in size to take up the space left behind by the Message Preview window.

Display Font Size:
This setting allows you to adjust the font size within the Direct Read Mail interface.

"New Mail" Sound File:
The setting allows you to select a sound file (wav, mp3, etc.) that will play any time your mailbox receives new email messages. To disable this feature, simply leave the settings field blank.
