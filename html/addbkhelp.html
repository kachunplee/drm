Add New Contact:
This area allows you to create a new contact within your address book. This feature is typically used to link a contact name to one or more email addresses, so that you don't have to type the address information every time you send email to those recipients. As an example, if you frequently send email to the same group of friends...

friend_1@aol.com
friend_2@earthlink.net
friend_3@college.edu
friend_4@someisp.com

...you can link those 4 addresses to a short contact name like "Friends". When you send email to that group of people in the future, you can simply address your message to "Friends" and our interface will automatically associate that contact name to the email addresses you specified.

Contact Name:
This is where you specify the contact name that will be linked to one or more email addresses. Using the example from above, "Friends" would be a contact name.

Email Address(es):
This is where you reference the email address(es) you wish to link to a contact name. Using the example from above, the "Friends" contact name is linked to 4 email addresses, so when a message is addressed/sent to "Friends", it will be delivered to those 4 email addresses.
 
 Current Contacts:
This area provides an overview of each contact listing you've created, and allows you to edit or delete a specific item from that list.

Contact:
This column notes the contact name for each of the listings you've created.

Address(es):
This column notes the email address(es) linked to a particular contact name.

Options Floating Menu:
When you move your cursor over the "Options" button, a floating menu will appear with the following options:

  • Edit Contact - This option will display the current settings for a specific contact listing, and enables you to edit any of the details within that listing.
     
  • Delete Contact - This option will delete a specific contact listing.
